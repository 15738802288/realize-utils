# best-utils

一个一致性、模块化、高性能的 JavaScript 实用工具库。

## 介绍:bulb:

一个封装了很多的高性能函数第三方工具库。如：数组、对象、函数等重要的一些功能性方法。

> 高性能且可兼容的

best-utils 不仅有简单并常用的函数，也有很多高性能且兼容的函数。

> 轻量容易

使用方便，配置简单，上手容易。

## 安装:wrench:

``` shell
npm i best-utils
```

## 快速上手:key:

```js
import { chunk } from 'best-utils'
let newChunk = chunk([1, 2, 3, 4, 5, 6, 7, 8, 9], 2)

console.log(newChunk); 
// [ [1, 2], [3, 4], [5, 6], [7, 8], [9] ] 
```
